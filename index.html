It's a GO for your own virtual pet!

Who loves virtual pets? Umm. You do. That's who. That's why you're here -- to produce your own artificially intelligent super pet! Well, let us not get too ahead of ourselves. Would you settle for a friendly rhino, giraffe, monkey, or other creation? If so, you're in luck.

Because not only will you be able to feed and care for your digital creation, you'll learn something even more amazing: how to create one! We'll be using Meteor, which is a super awesome way to make apps, and this might be your first app ever! It's really easy to get started. And if you make some improvements, at the end you can even make this project even better by sharing your improvements with everyone!

Are you ready? Here we go, young programmer.

There are some things you have to do to have to get rolling:

  1. Create an account at nitrous.io. This is easy. Click over to nitrous.io and click sign up. Nitrous is great because it lets us do all our application development in a computer that lives "in the cloud" -- which is just a fancy way of saying, it's a computer we can access via our web browser, no matter where we are.
    • Once you're logged in, make a new node.js box. Here is a quick video
  2. Once your box is created, you'll need to install Meteor and then create the directory that holds your first app.
    • Type (1) parts install meteor in the console, then (2) cd workspace and (3) meteor create myawesomepet and you're all done. Here is a quick video.
  3. Next, you'll need to install a program called atmosphere. Don't worry about all the details, just follow the below:
    • Type npm install -g meteorite in the command line, then cd myawesomepet, then mrt add meteorpets Here is a quick video

Woah. You made it! Your computer in the cloud is ready!
